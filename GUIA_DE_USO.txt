PASO A PASO PARA ENVIAR UN MENSAJE:

1) CREAR NUEVO MENSAJE
   ┌─────────────────────────────────────┐
   │ → Mensajes > Nuevo Mensaje          │
   │ → Seleccionar tipo:                 │
   │   • Email                           │
   │   • WhatsApp                        │
   │   • SMS                             │
   └─────────────────────────────────────┘

2) COMPLETAR DETALLES
   ┌─────────────────────────────────────┐
   │ → Asunto (para email)               │
   │ → Contenido del mensaje             │
   │ → Variables personalizadas          │
   │ → Archivos adjuntos (si aplica)     │
   └─────────────────────────────────────┘

3) SELECCIONAR DESTINATARIOS
   ┌─────────────────────────────────────┐
   │ → Importar lista (CSV/Excel)        │
   │ → Seleccionar grupo existente       │
   │ → Agregar destinatarios manual      │
   └─────────────────────────────────────┘

4) PROGRAMAR ENVÍO (OPCIONAL)
   ┌─────────────────────────────────────┐
   │ → Fecha y hora específica           │
   │ → Envío recurrente                  │
   │ → Zonas horarias                    │
   └─────────────────────────────────────┘

5) CONFIRMAR Y ENVIAR
   ┌─────────────────────────────────────┐
   │ → Previsualizar mensaje             │
   │ → Verificar destinatarios           │
   │ → Confirmar envío                   │
   └─────────────────────────────────────┘
